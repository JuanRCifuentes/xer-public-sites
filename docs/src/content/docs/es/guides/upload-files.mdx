Esta guía explica cómo subir archivos a un proyecto existente en XER.

## Formatos de archivo compatibles

XER admite los siguientes formatos de archivo para subir:

| Formato  | Extensión | Tamaño Máx. | Descripción                          |
|----------|-----------|-------------|--------------------------------------|
| XER      | `.xer`    | 50 MB       | Archivos de exportación de Primavera P6 |
| XML      | `.xml`    | 50 MB       | Archivos XML de Microsoft Project    |
| MPP      | `.mpp`    | 50 MB       | Archivos de Microsoft Project        |

## Cómo subir archivos

<Steps>

1. **Abre tu proyecto**

   Navega a **Proyectos** y selecciona el proyecto donde deseas subir archivos.

## Solución de problemas

### La carga falla o se queda atascada

- Verifica tu conexión a internet
- Comprueba que el formato del archivo sea compatible
- Asegúrate de que el archivo no exceda el tamaño máximo
- Intenta refrescar la página y subir de nuevo

### El archivo es rechazado después de la carga

- El archivo puede estar corrupto. Intenta exportarlo nuevamente desde la aplicación de origen
- Verifica que el archivo no esté protegido con contraseña
