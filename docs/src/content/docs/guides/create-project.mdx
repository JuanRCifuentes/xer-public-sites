This guide walks you through creating a new project in XER.

## Prerequisites

Before creating a project, make sure you have:

- An active XER account with the appropriate permissions
- Your project details ready (name, description, dates, etc.)

## Steps to create a project

<Steps>

1. **Navigate to the Projects section**

   From the main dashboard, click on **"Projects"** in the left sidebar navigation menu.

   {/* TODO: Add screenshot of the sidebar with Projects highlighted */}

2. **Click "New Project"**

   In the top-right corner of the Projects page, click the **"New Project"** button.

   {/* TODO: Add screenshot of the New Project button */}

3. **Fill in the project details**

   Complete the required fields in the project creation form:

   | Field             | Description                                      | Required |
   |-------------------|--------------------------------------------------|----------|
   | Project Name      | A descriptive name for your project               | Yes      |
   | Description       | A brief summary of the project's purpose          | No       |
   | Start Date        | The planned start date                            | Yes      |
   | End Date          | The planned end date                              | Yes      |
   | Project Type      | Select the type of project from the dropdown       | Yes      |

   {/* TODO: Add screenshot of the project creation form */}

4. **Configure project settings**

   Optionally, configure additional settings such as:

   - Team members and roles
   - Notification preferences
   - Access permissions

   {/* TODO: Add screenshot of the project settings section */}

5. **Save the project**

   Click **"Create"** to save your new project. You will be redirected to the project dashboard.

</Steps>

<Aside type="tip">
  You can always edit the project details later from the project settings page.
</Aside>
