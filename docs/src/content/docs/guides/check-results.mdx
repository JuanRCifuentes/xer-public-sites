This guide explains how to review and analyze the results generated after processing your files in XER.

## Results overview

After your files are uploaded and processed, XER generates a set of results that include:

- **Schedule Analysis** — Review of the project schedule with key metrics
- **Data Quality Check** — Validation of data integrity and completeness
- **Warnings & Errors** — Issues found during processing
- **Summary Dashboard** — Visual overview of your project data

## Filtering and sorting results

You can filter and sort results to find specific information:

- **Filter by status** — Show only items with warnings or errors
- **Filter by date** — Narrow down results to a specific time range
- **Sort by metric** — Order results by any column header
- **Search** — Use the search bar to find specific activities or items

## Comparing results

If you have uploaded multiple versions of a file, you can compare results:

<Steps>

1. Go to the **"Results"** tab
2. Click **"Compare"**
3. Select the two versions you want to compare
4. Review the differences highlighted in the comparison view

</Steps>
