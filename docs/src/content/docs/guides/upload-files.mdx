This guide explains how to upload files to an existing project in XER.

## Supported file formats

XER supports the following file formats for upload:

| Format   | Extension | Max Size  | Description                          |
|----------|-----------|-----------|--------------------------------------|
| XER      | `.xer`    | 50 MB     | Primavera P6 export files            |
| XML      | `.xml`    | 50 MB     | Microsoft Project XML files          |
| MPP      | `.mpp`    | 50 MB     | Microsoft Project files              |

<Steps>

1. **Open your project**

## Troubleshooting

### Upload fails or gets stuck

- Check your internet connection
- Verify the file format is supported
- Ensure the file does not exceed the maximum size
- Try refreshing the page and uploading again

{/* TODO: Add more troubleshooting items based on common issues */}

### File is rejected after upload

- The file may be corrupted. Try exporting it again from the source application
- Check that the file is not password-protected
