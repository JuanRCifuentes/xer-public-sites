# Landing

Marketing landing site built with Astro 6, Tailwind CSS v4, and deployed on Cloudflare Workers via Wrangler.

## Project Structure

```
src/
├── components/
│   ├── Navigation.astro     # Sticky nav with language switcher flyout (el-popover)
│   ├── Header.astro         # Page header (title + description)
│   ├── Footer.astro         # Footer with columns, newsletter, social links
│   ├── HeroSection.astro    # Landing hero with badge, CTAs, screenshot
│   ├── LogoCloud.astro      # Partner logos grid
│   ├── FeatureCards.astro   # 3-column feature cards with icons
│   ├── FeatureGrid.astro    # Screenshot + 6 mini-features grid
│   ├── StatsSection.astro   # 4-column stats
│   ├── CtaSection.astro     # Call-to-action section
│   └── Icon.astro           # SVG icon renderer (outline + solid maps)
├── data/
│   ├── en/                  # English content
│   │   ├── home.ts          # Home page data (hero, features, stats, cta)
│   │   ├── layout.ts        # Nav links, footer columns, newsletter
│   │   └── blog.ts          # Blog page labels
│   ├── es/                  # Spanish content (same structure as en/)
│   └── blog/                # Blog posts (content collection)
│       ├── en/              # English posts (.md)
│       └── es/              # Spanish posts (.md)
├── layouts/
│   └── BaseLayout.astro     # Root layout with SEO meta, Navigation, Footer
├── pages/
│   ├── index.astro          # Root redirect → /en/
│   └── [locale]/
│       ├── index.astro      # Home page (dynamic per locale)
│       └── blog/
│           ├── index.astro      # Blog listing
│           └── [...slug].astro  # Blog post with JSON-LD
├── utils/
│   └── i18n.ts              # Locale helpers (locales, names, path utils)
├── content.config.ts        # Blog collection schema (Zod)
└── styles.css               # Tailwind + @tailwindcss/typography
```

## i18n

All URLs are prefixed with the locale (`/en/`, `/es/`). Root `/` redirects to `/en/`.

- Locales defined in `src/utils/i18n.ts` and `astro.config.mjs`
- Page content loaded via dynamic imports from `src/data/{locale}/`
- Blog posts use a `locale` field in frontmatter, stored in `src/data/blog/{locale}/`
- Language switcher in Navigation preserves the current path

### Adding a new language

1. Add the locale to `locales` and `localeNames` in `src/utils/i18n.ts`
2. Add to `astro.config.mjs` → `i18n.locales` and `sitemap.i18n.locales`
3. Create `src/data/{locale}/` with `home.ts`, `layout.ts`, `blog.ts`
4. Add blog posts in `src/data/blog/{locale}/`

## Blog

Uses Astro content collections with the `glob` loader. Posts are markdown files in `src/data/blog/{locale}/`.

### Frontmatter schema

```yaml
title: "Post Title"
description: "Short description"
pubDate: 2026-03-10
author: "Author Name"
image: "https://..."   # optional
locale: "en"           # must match the folder
```

## SEO

- **Meta tags**: `description`, canonical URL, Open Graph, Twitter Cards (via BaseLayout props)
- **hreflang**: Alternate links for all locales + `x-default`
- **Sitemap**: Auto-generated by `@astrojs/sitemap` with i18n mapping
- **robots.txt**: In `public/`
- **JSON-LD**: Article structured data on blog posts
- **Site URL**: `https://xer.juanrcifuentes.com` (update in `astro.config.mjs` and `public/robots.txt`)

## Commands

| Command         | Action                                  |
| :-------------- | :-------------------------------------- |
| `bun install`   | Install dependencies                    |
| `bun dev`       | Start dev server at `localhost:4321`    |
| `bun run build` | Build production site to `./dist/`      |
| `bun preview`   | Preview build locally                   |

**Note**: Use `bun run build`, not `bun build` (which invokes Bun's bundler instead of Astro).

## Tech Stack

- [Astro 6](https://astro.build) with i18n routing (`prefixDefaultLocale: true`)
- [Tailwind CSS v4](https://tailwindcss.com) via `@tailwindcss/vite`
- [`@tailwindcss/typography`](https://github.com/tailwindlabs/tailwindcss-typography) for blog prose styling
- [`@tailwindplus/elements`](https://tailwindplus.com) for dialog, popover components
- [`@astrojs/sitemap`](https://docs.astro.build/en/guides/integrations-guide/sitemap/) for sitemap generation
- [Cloudflare Workers](https://workers.cloudflare.com) via Wrangler
